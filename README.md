### OAuth & Permissions

- Add below **Bot Token Scopes** in **Scope**

  - chat.write
  - chat.write.public

- Add below **User Token Scopes** in **Scope**

  - channels.history
  - channels.read

- **Install App**

- Copy `OAuth Access Token` and use as `USER_OAUTH_TOKEN`

- Copy `Bot User OAuth Access Token` and use as `BOT_OAUTH_TOKEN`
